      This pattern is available as a free Ravelry download

      A boyish/unisex version of the Brooklynn Jacket. The Zach jacket is knit top down from the collar. An optional hood is knit on afterwards. It also features almost seam-free pockets.

      Designed for variagated or semi-solid yarns, the Zach Jacket is an essential part of every child’s winter wardrobe.

      This pattern is written for worsted weight yarns. It is written for sizes nb/15” to 8y/26.5” chest. There is an eBook that includes both worsted and bulky weight versions.

      Worsted - 20st/4” gauge - Approximate yardage required:

      Main - 200(225, 250, 275, 300, 350, 400) 450, 500, 550, 600, 700yards.
      Contrast - 65(75, 85, 100, 100, 100, 125) 125, 150, 175, 200, 200yards.
      Hood - 75(75, 100, 100, 100, 125, 125) 150, 175, 200, 225, 250yards
