      This pattern is available for free.

      This is a really cute little baby blanket

      Just 15” x 15” - for a Premature or Angel baby - or doll

      Fun to knit !

      Requires ~
      DK yarn - around 70g
      4mm needles

      Tension/Gauge ~
      24sts = 4” (10cm)

      Easy to adjust for bigger sizes -
      Stitches given for 18”, 24” and 30” baby blankets

      So sweet

      PLEASE NOTE **
      There was originally an error in this pattern ~
      The purl rows should read
      S1, k5, purl to last 6sts, k6

      Sorry. I have corrected the instructions now.
